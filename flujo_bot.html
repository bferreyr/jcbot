Flujo del Asistente Virtual

Descubre cómo funciona nuestro sistema inteligente de atención al cliente paso a paso.

📱
1. Interacción del Cliente

Mensaje de WhatsApp

El cliente envía un mensaje, pregunta o solicitud a tu número de WhatsApp Business, igual que lo haría con cualquier otra persona.

🌐
2. Recepción (Webhook)

Conexión con el Servidor

Meta (WhatsApp API) detecta el mensaje y lo envía de forma instantánea y segura a nuestro servidor inteligente a través de un Webhook.

🗄️
3. Contexto e Identificación

Base de Datos

El sistema identifica al cliente por su número, guarda el mensaje y recupera el historial de la conversación para entender el contexto (de qué venían hablando).

📊
4. Información en Tiempo Real

Google Sheets

El bot consulta una hoja de cálculo de Google (actualizable por ti) para obtener la información fresca del negocio: precios actuales, horarios, dirección y promociones.

🧠
5. Procesamiento Inteligente

Inteligencia Artificial (Claude 3)

Se envía a la IA avanzada: el mensaje actual + el historial de chat + los datos del negocio. La IA analiza todo y redacta una respuesta perfecta, natural y amigable.

✉️
6. Respuesta Automática

Envío al Cliente

La respuesta generada por la IA se envía de vuelta al cliente mediante la API de WhatsApp, logrando una atención rápida, precisa y 24/7.
