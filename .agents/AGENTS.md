# Reglas de Comportamiento del Proyecto

## Flujo de Trabajo y Git
- Al finalizar cualquier tarea, plan de implementación o corrección solicitada por el usuario, DEBES ejecutar automáticamente los comandos de Git para agregar los cambios, crear un commit con un mensaje descriptivo y realizar un push al repositorio en GitHub, a menos que el usuario indique explícitamente lo contrario.
